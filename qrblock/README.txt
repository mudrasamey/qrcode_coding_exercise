INTRODUCTION
------------

This is a very simple module that adds a QR code block to the product page.

REQUIREMENTS
------------

No special requirements.

INSTALLATION
------------

Install as you would normally install a  Drupal module. See:
https://www.drupal.org/docs/8/extending-drupal/installing-contributed-modules
for further information.

CONFIGURATION
------------

The QR Block should be placed in the second sidebar for the product pages.
In case you want to change this setting you can do so from the Block Layout.
i.e Home-> Administration-> Structure-> Block layout.

On the configure page on this block the text configuration is provided.
